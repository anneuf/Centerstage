# Centerstage
Centerstage Teleprompter System

Based on Raspian

Initial ToDos:

 1) Setup Raspian using Noobs
 2) Run raspi-config
    a) Disable Overscan
    b) Enable i2c extension
    c) Set locale
    d) Set Boot to console
 3) Enter WPA-credentials in conf/install_wpa.sh
 4) Run install.sh with root privileges
